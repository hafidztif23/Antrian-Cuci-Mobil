ANTRIAN CUCI MOBIL

Program ini memiliki fungsi untuk membuat ilustrasi antrian di car wash dengan menggunakan linked list dan queue. Program ini diciptakan oleh kelompok 5 yang beranggotakan 3 orang, yaitu Firlyansyah Putra Pratama (231511010), Hafidz Zaenul Ahkam (231511012) dan Moch Rizky Taufiqurrahman (231511018). Di dalam program ini terdapat 2 stasiun cuci basah, 2 stasiun pengering, waiting list untuk cuci basah dan waiting list untuk cuci kering.

Alur dari program Antrian Cuci Mobil:
1. Ketika ada mobil yang datang, program akan meminta untuk menginputkan jenis mobil dan plat nomor dari mobil.
2. Jika stasiun cuci basah kosong, maka mobil akan dialokasikan ke stasiun tersebut. Sedangkan, jika stasiun cuci basah masih terdapat mobil yang lain, maka mobil akan dialokasikan ke waiting list cuci basah.
3. Setelah cuci basah selesai, maka mobil akan dialokasikan ke stasiun pengering.
4. Jika stasiun cuci kering kosong, maka mobil akan dialokasikan ke stasiun tersebut. Sedangkan, jika stasiun pengering masih terdapat mobil yang lain, maka mobil akan dialokasikan ke waiting list cuci kering.
5. Ketika proses penyucian mobil selesai, maka mobil akan didealokasikan dari stasiun cuci kering.

Fitur program:
1. Add queue: Fitur untuk mengalokasikan mobil ke stasiun cuci basah atau waiting list cuci basah.
2. Selesaikan pengerjaan: Fitur untuk menyelesaikan pencucian di stasiun.
3. Tampilkan daftar mobil: Fitur untuk menampilkan informasi lengkap dari mobil-mobil yang terdapat di stasiun dan waiting list
4. Cari mobil: Fitur untuk mencari mobil berdasarkan plat nomor yang diinput. Jika ditemukan maka program akan menampilkan informasi lengkap dari mobil tersebut.
5. Tampilkan log: Fitur untuk menampilkan riwayat dari mobil yang telah dicuci.
6. Batalkan antrian: Fitur untuk membatalkan antrian. Fitur ini hanya bisa dilakukan di waiting list cuci basah.
7. Pembayaran: Fitur untuk melakukan pembayaran jasa cuci mobil.
8. Estimasi dikerjakan: Fitur untuk menampilkan kapan mobil yang berada di waiting list akan dikerjakan.
